# Cubematic
Cubematic is a collection of minecraft server plugins and related code.
The plugins target Paper servers or compatible forks.

# Cubematic's plugins
- Agent: In-game LLM powered chatbot
- Automation: Adds more automation features, such as block breaker, block placer
- Dreams: TBD
- HUD: Coordinates and heading display
- InTheSky: Skyblock style plugin to add more content to a skyblock like gamemode
- Portals: Adds in-game teleportation portals
- Runtime: Runtime library for the plugins to support kotlin etc.
